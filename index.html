Suggestions & Next Features

  • QR types: vCard, Wi‑Fi, Email, SMS, Geo, Calendar, and bulk generation (CSV upload).
  • Branding: templates, gradient/eye shapes, logo safe‑zone checker, vector (SVG) export with outlines.
  • Short links & analytics: integrate a link shortener with scan analytics (UTM, device, geo).
  • Scanner enhancements: barcode formats (EAN/UPC/PDF417), torch toggle, file‑scan batch mode.
  • Background remover: add API mode (e.g., remove.bg) for general objects; queue + batch.
  • Workspace: project folders, recent items, version history, one‑click social export sizes.
  • Installable PWA: offline QR generator & scanner; camera permissions remembered.
  • Auth & billing: free tier limits, credits for BG removals, webhooks, and REST API.

Privacy: Everything runs locally in your browser. Camera frames are processed on‑device. If you later enable an external API for background removal, those images will leave the device per that provider’s policy.
